## Getting Started

### Prerequisites

- NPM
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/renatozr/weather-app.git
   ```
2. Enter the directory
   ```sh
   cd weather-app
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the project
   ```sh
   npm run dev
   ```
